Questions, answered

Before you queue up.

Which games does GameFeed work with?＋

GameFeed works best with windowed and borderless-fullscreen games. Hardware-exclusive fullscreen and some anti-cheat systems can block third-party overlays.

Do I sign in through GameFeed?＋

Yes. Each platform opens its own website inside GameFeed's embedded browser, and the local browser profile keeps your session available. Sign in before opening your game because some games capture keyboard input and can prevent typing into login screens while they are running.

Can I make the overlay transparent?＋

Yes. You can adjust opacity, enable click-through, freeze the size, and save the placement that fits your setup.

Does GameFeed support custom shortcuts?＋

Yes. Previous, next, mute, show/hide, and click-through can use supported keyboard keys or Mouse 4/5.

Is the GameFeed download safe?＋

Yes. The download button uses GameFeed’s official GitHub Release and the installer contains only GameFeed and its required runtime files—no bundled offers or extra apps. The current release is not yet code-signed, so Windows may show an “unrecognized app” SmartScreen message even though the file came from the official source.

Simple Windows setup

Download, install, start scrolling.

The new installer handles the app files and shortcuts for you. There is no ZIP folder to extract.

  1. 01

    Download the installer

    Choose “Download GameFeed Setup” and open GameFeed-Setup.exe from your browser’s Downloads list.

  2. 02

    Follow the setup window

    Choose Install. GameFeed is added to your Start Menu and a desktop shortcut is created automatically.

  3. 03

    Launch GameFeed

    If SmartScreen appears, confirm the filename is GameFeed-Setup.exe, then select More info → Run anyway. This notice appears because the early release is not yet code-signed.
